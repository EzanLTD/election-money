About Election Money

Interested in money in Illinois politics? This site makes availablebulk downloads of Illinois campaign contributions and expenditures from the Illinois State Board of Elections going back to 1994.

Understanding the data

Interested in learning more about campaign finance disclosure in Illinois? Here are a few useful resources:

  • Official Guide to Campaign Disclosure (PDF)
  • Illinois Campaign Disclosure Act (PDF)
  • Illinois Campaign Financing Act - Contribution Limits (PDF)
  • Campaign Disclosure Glossary A - E
  • Frequently Asked Questions about Campaign Disclosure

Team and credits

This is a DataMade project done in collaboration with the Illinois Campaign for Political Reform that was spawned at a meeting held in Chicago with the Sunlight Foundation. We built this site to increase the access to campaign contribution data for journalists, activists and curious citizens.

Code

All the code for this site is open source. It consists of a scraper and this website to display the data. It was built with:

  • Flask - python web framework
  • Amazon S3 - file storage
  • Bootstrap - HTML and CSS scaffolding
  • Datatables - searchable, sortable HTML tables
  • Heroku - website hosting

All the code for this project is available on Github:

  • election-money - this website
  • campaign-finance_state_IL - Illinois State Board of Elections website scraper
